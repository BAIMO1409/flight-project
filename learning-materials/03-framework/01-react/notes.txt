=======================================================
                  React 学习笔记
=======================================================

一、核心知识点

1. React基础
   - 简介: 用于构建用户界面的JavaScript库
   - 特点: 组件化、虚拟DOM、单向数据流
   - 安装: npm install react react-dom

2. JSX语法
   - 简介: JavaScript XML，在JavaScript中写HTML
   - 特点: 类HTML语法、支持表达式嵌入、自动转义
   - 注意: 标签必须闭合、class改为className、for改为htmlFor

3. 组件
   - 函数式组件: function Component() {}
   - 类组件: class Component extends React.Component {}
   - 无状态组件: 仅展示，无state
   - 有状态组件: 有state和生命周期

4. Props
   - 简介: 父组件传递给子组件的数据
   - 特点: 只读、单向传递
   - 默认值: defaultProps
   - 类型检查: PropTypes

5. State
   - 简介: 组件内部的状态数据
   - 特点: 可变、触发重渲染
   - 更新方式: setState()（类组件）、useState()（函数组件）

6. Hooks
   - useState: 状态管理
   - useEffect: 副作用处理
   - useContext: 上下文传递
   - useReducer: 复杂状态管理
   - useCallback: 回调函数缓存
   - useMemo: 计算结果缓存

7. 生命周期（类组件）
   - 挂载: constructor → componentWillMount → render → componentDidMount
   - 更新: componentWillReceiveProps → shouldComponentUpdate → componentWillUpdate → render → componentDidUpdate
   - 卸载: componentWillUnmount

8. 事件处理
   - 绑定方式: 箭头函数、bind绑定、类属性箭头函数
   - 事件对象: React事件（合成事件）
   - 参数传递: 闭包、data属性

9. 条件渲染
   - if/else
   - 三元表达式
   - &&运算符
   - 条件渲染组件

10. 列表渲染
    - map()方法
    - key属性（必须唯一）
    - 遍历对象

二、重点难点

1. Hooks的使用规则
   - 只能在函数组件顶层调用
   - 不能在循环、条件、嵌套函数中调用
   - 自定义Hook必须以use开头

2. useEffect的依赖数组
   - 空数组: 仅挂载时执行一次
   - 无依赖数组: 每次渲染都执行
   - 有依赖: 依赖变化时执行

3. 虚拟DOM与diff算法
   - 虚拟DOM: JavaScript对象表示DOM
   - Diff算法: 高效比较新旧虚拟DOM
   - Key的重要性: 帮助识别列表项变化

三、学习心得

- React的组件化思想让代码更模块化
- Hooks大幅简化了函数组件的开发
- JSX让UI开发更直观
- 单向数据流让状态管理更清晰

四、关键概念总结

- 虚拟DOM: 用JavaScript对象描述DOM结构
- Diff算法: 比较新旧虚拟DOM并最小化DOM操作
- 组件: 可复用的UI单元
- 状态提升: 将共享状态提升到共同父组件
- Context: 跨层级传递数据

=======================================================
                    CSS 学习笔记
=======================================================

一、核心知识点

1. CSS 语法
   - 选择器 { 属性: 值; }
   - 注释: /* 注释内容 */
   - CSS优先级: !important > 内联样式 > ID选择器 > 类选择器 > 元素选择器 > 通配符

2. 选择器
   - 元素选择器: p, div, span
   - 类选择器: .class-name
   - ID选择器: #id-name
   - 属性选择器: [attribute], [attribute=value]
   - 伪类: :hover, :active, :focus, :nth-child(n)
   - 伪元素: ::before, ::after, ::first-line

3. 盒模型
   - content: 内容区域
   - padding: 内边距
   - border: 边框
   - margin: 外边距
   - box-sizing: content-box（默认）/ border-box

4. 浮动（float）
   - left/right/none
   - 清除浮动: clear: both
   - 浮动导致的父元素塌陷问题

5. 定位（position）
   - static: 默认值，文档流定位
   - relative: 相对定位，相对于自身位置偏移
   - absolute: 绝对定位，相对于最近定位祖先
   - fixed: 固定定位，相对于视口
   - sticky: 粘性定位，滚动到阈值时固定

二、重点难点

1. CSS优先级计算
   - ID选择器: 100
   - 类/伪类/属性选择器: 10
   - 元素/伪元素选择器: 1
   - 内联样式: 1000
   - !important: 最高优先级

2. 浮动布局
   - 浮动元素脱离文档流
   - 父元素高度塌陷解决方案:
     * 清除浮动（clearfix）
     * BFC（块级格式化上下文）
     * 父元素overflow: hidden

3. 定位层级
   - z-index控制堆叠顺序
   - 只有定位元素（非static）才受z-index影响

三、学习心得

- CSS是页面的外衣，决定了视觉呈现效果
- 选择器的优先级规则很重要，避免样式冲突
- 盒模型是布局的基础，border-box能简化计算
- 浮动和定位需要理解元素在文档流中的行为

四、关键概念总结

- 文档流: 元素在页面中的默认排列顺序
- BFC: 块级格式化上下文，独立的渲染区域
- 层叠上下文: 元素的堆叠层级关系
- 继承: 子元素继承父元素的某些样式属性

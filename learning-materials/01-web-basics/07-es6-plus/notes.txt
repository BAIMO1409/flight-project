=======================================================
                  ES6+ 学习笔记
=======================================================

一、核心知识点

1. let/const
   - let: 块级作用域，变量可以重新赋值
   - const: 块级作用域，变量不可重新赋值（但对象属性可修改）
   - 区别于var: 不存在变量提升、暂时性死区

2. 解构赋值
   - 对象解构: const { name, age } = person
   - 数组解构: const [a, b, ...rest] = arr
   - 默认值: const { name = 'default' } = obj
   - 重命名: const { name: userName } = obj

3. 模板字符串
   - 语法: `Hello ${name}`
   - 支持多行字符串
   - 支持表达式嵌入

4. 箭头函数
   - 语法: (param) => expression
   - 特性: 没有自己的this、arguments、super
   - 适合用于回调函数

5. Promise
   - 三种状态: pending（进行中）、fulfilled（已成功）、rejected（已失败）
   - 方法: then()、catch()、finally()
   - 静态方法: Promise.all()、Promise.race()、Promise.allSettled()

6. async/await
   - async函数返回Promise
   - await等待Promise解决
   - try/catch处理错误

7. Class
   - 语法: class Name { constructor() {} method() {} }
   - 继承: extends
   - 方法: constructor、static、getter/setter
   - 私有字段: #field（ES2022）

8. 模块化
   - import: 导入模块
   - export: 导出模块
   - 默认导出: export default

9. 可选链（Optional Chaining）
   - 语法: obj?.prop、arr?.[0]、func?.()
   - 避免空指针异常

10. 空值合并运算符（Nullish Coalescing）
    - 语法: value ?? defaultValue
    - 仅在value为null或undefined时返回默认值

11. 展开运算符（Spread Operator）
    - 数组展开: [...arr1, ...arr2]
    - 对象展开: {...obj1, ...obj2}
    - 函数参数展开: func(...args)

12. 剩余参数（Rest Parameters）
    - 语法: function(...args) {}
    - 收集所有剩余参数为数组

二、重点难点

1. 箭头函数的this指向
   - 箭头函数没有自己的this
   - this继承自外层作用域的this
   - 不适合作为构造函数和方法

2. async/await的错误处理
   - 使用try/catch捕获错误
   - Promise.all()中任一Promise失败会导致整体失败
   - Promise.allSettled()等待所有Promise完成

3. Class与原型链的关系
   - Class是原型链的语法糖
   - constructor对应构造函数
   - prototype方法会被所有实例共享

三、学习心得

- ES6+大幅提升了JavaScript的开发效率
- 箭头函数简化了回调函数的写法
- async/await让异步代码更像同步代码
- 解构赋值和展开运算符让数据处理更简洁

四、关键概念总结

- 块级作用域: let/const的作用域在{}内
- 暂时性死区: let/const声明前无法访问
- Promise链: 通过then()串联多个异步操作
- 异步函数: async函数返回Promise，await暂停执行
